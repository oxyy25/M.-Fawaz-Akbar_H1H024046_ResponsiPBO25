pada folder class :
- pokemon.php         : class abstrack dari pokemon yang berisi stasitik pokemon dan perhitungan kenaikan level pokemon
- figthingpokemon.php : turunan dari class abstrack pokemon
- trainingHistroy     : tempat menyimpan hasil latihan pokemon

- index.php     : halaman awal untuk melihat stasitik pokemon

- History.php   : halaman melihat riwayat latihan pokemon
